# CI Dark

![preview](https://raw.githubusercontent.com/chuling/vim-equinusocio-material-preview/master/ci_dark.png)

## Usage

iTerm color preset: [ci_dark.itermcolors](term/ci_dark.itermcolors)

```viml
syntax on
set termguicolors
colorscheme ci_dark
let g:lightline = {
  \ 'colorscheme': 'ci_dark',
}
```
